# Security Policy

These are gameplay mods for Farming Simulator 25. They read game state and write their own settings and savegame data — they do **not** transmit anything off your machine.

## Reporting a concern
If you find a genuine security or data-loss issue (for example a mod corrupting saves or handling files unsafely):
- Open an issue on the affected repository, **or**
- Hold off on posting exploit details publicly until it is fixed.

## Supported versions
Only the **latest released version** of each mod is supported. Please update before reporting.
